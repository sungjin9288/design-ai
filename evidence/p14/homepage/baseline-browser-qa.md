- Preview: loopback MkDocs build before the approved homepage implementation.
- Desktop: 1440×900.
- Mobile: 390×844.
- Horizontal overflow: not observed in either captured viewport.
- First-value path: no prominent install or quickstart action appeared in the
  first mobile viewport; badges and the release-status paragraph led the page.
- Accessibility: canonical static inspection confirmed one unnamed search toggle
  input. Skip-link destination focus was not proven in the baseline capture.
- Console: the previous Pretendard CDN request returned 403. The local
  multilingual preview also requested `/ko/sitemap.xml`, which returned 404.
- Repeated interaction and reduced motion: not retained for the baseline and
  therefore left unverified.

The screenshots and static review are the evidence of record. This note does not
infer unobserved runtime behavior.
