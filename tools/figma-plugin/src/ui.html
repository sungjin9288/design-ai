Token importer

Paste a W3C DTCG / Style Dictionary JSON. Tokens become Figma Variables in the current file.

Format help

Each top-level key (color / spacing / typography / etc.) becomes a Variable Collection. Nested keys flatten to dot-separated names.

Supported types: color, dimension, number, boolean, string.

For colors: hex (#rgb / #rrggbb / #rrggbbaa). OKLCH not yet supported — convert to hex first.

{
  "color": {
    "brand": {
      "primary": {
        "50":  { "$value": "#F5F3FF", "$type": "color" },
        "600": { "$value": "#7C3AED", "$type": "color" }
      }
    },
    "semantic": {
      "primary": { "$value": "#7C3AED", "$type": "color" },
      "text-primary": { "$value": "#0F172A", "$type": "color" }
    }
  },
  "spacing": {
    "xs": { "$value": "4px", "$type": "dimension" },
    "sm": { "$value": "8px", "$type": "dimension" },
    "md": { "$value": "12px", "$type": "dimension" }
  }
}
